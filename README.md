# 💻 Estudo de Lógica de Programação com VisuAlg 3.0

Bem-vindo ao meu repositório de estudos diários! Aqui guardo a evolução do meu raciocínio lógico, utilizando **pseudocódigo** (Portugal) para resolver desafios e exercícios práticos.

## 🚀 Objetivo
O foco deste repositório é consolidar os fundamentos da programação, como:
- Variáveis e Tipos de Dados
- Operadores Aritméticos e Lógicos
- Estruturas Condicionais (`SE/ENTAO/SENAO`, `ESCOLHA/CASO`)
- Estruturas de Repetição (`PARA`, `ENQUANTO`, `REPITA`)
- Vetores e Matrizes (Arrays)

## 🛠️ Ferramenta Utilizada
Para rodar os algoritmos deste repositório, utilizo o **VisuAlg 3.0**.
- [Download do VisuAlg 3.0](http://visualg3.com.br) - Site oficial para descarregar a ferramenta.

## 📝 Como executar
1. Faça o download do [VisuAlg 3.0](http://visualg3.com.br).
2. Abra o software.
3. Copie o código de qualquer arquivo `.alg` deste repositório.
4. Cole no editor do VisuAlg.
5. Pressione `F9` para rodar o algoritmo!
